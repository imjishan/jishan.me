Here are some of the most commonly use crontab commands I use. Wanted to put it in one place to reference back to whenever.

### Open crontab file

<CodeBlock title="Terminal">
```bash
crontab -e
```
</CodeBlock>

### List cron jobs

<CodeBlock title="Terminal">
```bash
crontab -l
```
</CodeBlock>
